A PyTorch-based optimization model that generates optimal decklists for the One Piece Trading Card Game (OPTCG). Uses gradient descent with Gumbel-Softmax to handle discrete card selection, and applies multi-objective loss to simultaneously optimize across all data entries per card. The model analyzes card statistics and synergies to recommend competitive, well-balanced deck configurations. Primary back-end developer using Python for card data analysis and PyTorch for model architecture and training.
