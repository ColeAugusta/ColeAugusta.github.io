A multi-threaded client-server code repository application with both a command-line interface (CLI) and a Qt GUI client. Features encrypted passwords and files, concurrent client handling, path traversal security, and a custom TCP protocol over socketed connections. Compatible with Windows and Linux, and deployed on an AWS EC2 instance running Amazon Linux 2023. Served as primary full-stack developer using C++, socketed TCP connections, the Qt library for GUI development, and shell-based testing.
