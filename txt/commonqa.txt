A Graph Neural Network (GCN and GAT) trained on a Commonsense Knowledge Graph to answer natural-language common-sense questions. Built a PyTorch Geometric graph to construct a portion of the ConceptNet 5.1 knowledge base, with a multi-layer GCN architecture trained on a link prediction task over the graph. Semantic profiles of input nodes enable the model to learn language embeddings and produce answers. Features an in-console QA system that prompts the model to respond in real time. Primary back-end developer using Python, PyTorch Geometric for graph construction, and PyTorch for model training.
